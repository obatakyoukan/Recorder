ruby on railsを使い実装する・改善する．

 - ログイン機能 Clear

 - ユーザー削除機能 Clear
 - データベースの作成Books

 - Books 追加・消去機能の追加 Clear
 - 追加問題点: ユーザーログインしていない状態で　urlからshowで利用でアクセスできる．
